My projects

Here are some of my projects that I have done. I am looking forward to creating more projects in the future!

NF online shop

NF Online Shop is a full-stack e-commerce platform built with HTML, CSS, JavaScript, and Node.js on the backend. I used Firebase for user authentication and data storage and Amazon Web Services (AWS) for managing product images. The site supports both buyers and sellers. You’re welcome to try it out! To access the seller interface, simply add /seller.html to the URL and you’ll be redirected (please be mindful of the content you may publish).

Amazon

This project is a replica of the Amazon website, built using HTML, CSS, and JavaScript. Users can add items to the cart, and the total delivery price is automatically calculated. To calculate the delivery date options, I used the external library Day.js. For testing, I used the Jasmine testing framework to ensure that all price and date calculations work correctly.

Flower Store

This project is a simple one-page website for selling flowers. I focused a lot on the visual design to make it look appealing. The site was built using HTML, CSS, and JavaScript. It includes a price calculator and a form for users who want to collaborate in the future. I used Anime.js to animate the buttons and applied event delegation for switching images in the “Our Service” section. There's also a responsive burger menu for the mobile version.
